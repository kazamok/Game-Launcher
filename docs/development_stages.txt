WoW 런처 개발 단계

1. 기본 구조 [완료]
   - 프로젝트 구조 생성
   - 메인 창 설정
   - 배경 이미지 추가

2. 인터페이스 [완료]
   - 로고 및 내비게이션이 있는 헤더 생성
   - 상태 카드 추가
   - 뉴스 블록 개발
   - 실행 버튼이 있는 하단 패널 생성

3. 스타일링 [완료]
   - 카드 디자인 개발
   - 아이콘 생성 및 추가
   - 그라데이션 및 효과 설정
   - 애니메이션 추가

4. 설정 [완료]
   - 설정 창 생성
   - JSON에 설정 저장 기능 구현
   - 탭 추가 (게임, 그래픽, 애드온)
   - 게임 경로 선택 기능 구현

5. 실행 기능 [개발 중]
   - 게임 존재 여부 확인
   - 게임 경로 유효성 검사
   - 매개변수와 함께 게임 실행
   - 실행 오류 처리

6. 네트워크 상호 작용 [계획 중]
   - 서버 API에 연결
   - 서버 상태 가져오기
   - 온라인 플레이어 목록 로드
   - 서버에서 뉴스 가져오기

7. 업데이트 시스템 [계획 중]
   - 클라이언트 버전 확인
   - 업데이트 다운로드
   - 진행률 표시
   - 패치 설치

8. 애드온 [계획 중]
   - 애드온 관리자 개발
   - 사용 가능한 애드온 목록 로드
   - 애드온 설치 및 업데이트
   - 애드온 구성 관리

9. 최적화 [계획 중]
   - 성능 최적화
   - 오류 처리 개선
   - 로깅 추가
   - 메모리 사용 최적화

10. 테스트 및 디버깅 [계획 중]
    - 모듈 테스트
    - 통합 테스트
    - 사용자 인터페이스 테스트
    - 발견된 오류 수정

현재 상태: 게임 실행 기능 개발 중
다음 단계: 네트워크 상호 작용 구현
